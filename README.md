# 豆瓣小组管理助手 - Chrome插件

## Features:
### 封禁并删除
封禁发帖用户并转向删除帖子页面，简化了封禁操作

### 找出僵尸用户
在用户列表中标注出僵尸用户（名号为“[已注销]”的用户），方便清理
_Notice: 此功能不再被支持，由于豆瓣小组管理已经做了“扫描/清理已注销成员”功能_

### 快速退出小组
在“[我参加的小组](http://www.douban.com/group/mine)”页面快速退出小组
